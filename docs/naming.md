# Naming

## Official Name

The project and app name is **Gokan**.

The full mark is **Gokan 碁冠**, read as "Go crown."

## Usage

- Use **Gokan** in English UI, filenames, repository names, module names, bundle
  names, and command-line tools.
- Use **Gokan 碁冠** for branding moments: website mastheads, app about screens,
  release pages, and formal documentation.
- Avoid styling the everyday app name as `GoKan`; the intended word is one
  smooth name, not a Go-language pun.

## Rationale

Gokan is inspired by Japanese Go title culture without reusing an official title
such as Kisei, Meijin, Honinbo, Oza, Tengen, Gosei, or Judan. The name is short,
easy to pronounce in English, and still carries the title/crown feeling through
the full mark.
